智能灌溉机器人的设计初衷是针对于比赛的，但其中也有一些通用的技术可以用来实现不同的赛道或目的，比如说其他的智能小车设计。
如：使用编码电机进行PID速度环的控制，再者可以使用陀螺仪进行姿态的判断，等等你可以自行使用。
